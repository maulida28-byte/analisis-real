Sifat Aljabar dari Bilangan Real

Kita dapat memperoleh suatu bilangan dengan cara mengoperasikan suatu bilangan dengan bilangan yang lain. Secara formal, hal ini mengarah kepada pengertian operasi biner. Operasi biner pada himpunan A didefinisikan sebagai fungsi A x A ke A. Terdapat dua operasi biner pada bilangan real, yaitu penjumlahan (+) dan perkalian (∙). Operasi biner erat kaitannya dengan sifat aljabar, yaitu merupakan alat utama dalam analisis. Oleh karena itu, kita akan mempelajari sifat-sifat aljabar dari sistem bilangan real. Dari sifat-sifat tersebut, kemudian di turunkan sifat-sifat penting lainnya sebagai teorema.

  • Komutatif (Penjumlahan)

    a + b = b + a untuk setiap a,b ∈ R; (Sifat komutatif dari penjumlahan)

  • Asosiatif(Penjumlahan)

    ( a + b ) + c = a + ( b + c ) untuk setiap a,b,c ∈ R; (Sifat assosiatif dari penjumlahan)

  • Eksistensi elemen nol

    Terdapat 0 ∈ R sehingga 0 + a = a untuk setiap a ∈ R; (Eksistensi elemen nol)

  • Eksistensi elemen negatif

    Untuk setiap a ∈ R terdapat - a ∈ R sehingga a + ( - a ) = 0; (Eksistensi elemen negatif)

  • Komutatif (Perkalian)

    a x b = b x a untuk setiap a,b ∈ R; (Sifat komutatif dari perkalian)

  • Asosiatif(Perkalian)

    ( a x b ) x c = a x ( b x c ) untuk setiap a,b,c ∈ R; (Sifat assosiatif dari perkalian)

  • Eksistensi elemen satuan

    Terdapat elemen 1 ∈ R , 1 ≠ 0, sehingga 1 x a = a untuk semua a ∈ R; (Eksistensi elemen satuan)

  • Eksistensi invers

    Untuk setiap a ∈ R, a ≠ 0; terdapat elemen 1/a ∈ R sehingga a ∙ (1/a) = 1; (Eksistensi invers)

  • Distribusi(Perkalian terhadap Penjumlahan)

    a x ( b + c ) = ( a x b ) + ( a x c) untuk semua a,b,c ∈ R; (Sifat distribusi perkalian terhadap penjumlahan)

Kesembilan aksioma di atas dikenal dengan aksioma lapangan. Oleh karena itu, himpunan bilangan real dengan operasi biner penjumlahan dan perkalian merupakan suatu lapangan atas dirinya sendiri.

Teorema berikut ini akan menjamin eksistensi bilangan 0 dan 1.Untuk menyederhanakan penulisan, selanjutnya a∙b cukup ditulis dengan ab.

  • Jika z dan a unsur di R sehingga z + a = a maka z = 0

      (-a) + 0 = -a.

    4. Oleh karena itu, dapat disimpulkan bahwa b = -a.

  • Jika a, b ∈ R sehingga a × b = 1, maka b = 1/as

    • Bukti:

    1. Diketahui bahwa ab = 1. Dengan mengalikan 1/a pada kedua ruas,

      ab · (1/a) = 1 · (1/a).

    2. Jika kita aplikasikan (M2), (M4), dan (M3) pada ruas kiri, maka diperoleh:

    • Penyelesaian
